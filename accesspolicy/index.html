Remote & VPN Access Policy

We at the Mac Consulting Group are committed to the privacy and confidentiality of your information. However, there are times that we cannot physically be at your office. With remote support we can access your computer/network from where ever we are. Below is an explanation of how this happens and what you are agreeing to by submitting this form.

The following statements apply to the implementation of Remote Access and VPN Access that allows direct connections to your company’s network from an outside location.

  • Remote Access is direct remote access to a computer system through use of remote support software such as:  Apple Remote Desktop, Screen Sharing, Mac HelpMate, TeamViewer, and Microsoft Remote Desktop Connection.
  • A Virtual Private Network (VPN) is a private network that is formed by joining two or more nodes together through an encrypted IP tunnel over the internet. This tunnel treats both end points as if they were directly joined on a local area network (LAN). A VPN connection can be be created a number of ways, for example: router to router connections, router to client connections, and server to client connections.
  • All client computer and VPN password and setup information is stored in encrypted form.
I grant Mac Consulting Group, Inc. access to our computer systems, networks, usernames and passwords on an as needed basis. I understand that this granted access includes all technical staff, including but not limited to: technicians, consultants, and contractors operating through Mac Consulting Group, Inc.
